# Lesson 7 – JavaScript Basics (Loops)

This project introduces loops in JavaScript to perform repeated actions.

## Features
- Paragraph with id `displayText` updates dynamically
- Button `clickMeBtn` updates text with a message
- Button `decisionBtn` uses `if/else` to choose between messages
- Button `repeatBtn` demonstrates a `for` loop to show numbers 1–5

## Files included
- index.html – Webpage structure
- script.js – JavaScript logic
- README.md – Project explanation
